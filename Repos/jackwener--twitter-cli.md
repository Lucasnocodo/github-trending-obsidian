# twitter-cli

**1.4k** stars · **275** stars/天 · 建立 5 天前 · Python · Apache-2.0

> [!summary] 一句話摘要
> twitter-cli 是一個用於 Twitter/X 的命令行介面。

## 專案簡介

twitter-cli 提供了一個命令行介面，讓使用者可以在終端機中閱讀 Twitter/X 的時間線、書籤和用戶資料，無需 API 金鑰。這個工具的特點是簡單易用，適合喜歡在終端機中操作的使用者。與其他 Twitter 客戶端相比，twitter-cli 不需要繁瑣的設置，直接可以使用，讓使用者能快速上手。對於喜歡命令行操作的 Twitter 用戶來說，這是一個值得嘗試的工具。

## 重點功能

- 無需 API 金鑰即可使用。
- 支持查看時間線、書籤和用戶資料。
- 簡單的命令行介面，易於上手。

## 快速開始

使用 pip 安裝：`pip install twitter-cli`。,運行命令：`twitter-cli timeline` 開始查看時間線。

## 為什麼值得關注

> [!tip] 爆紅原因
> 作者的開發背景讓這個工具具備良好的設計和使用體驗，並且滿足了許多喜歡命令行操作的用戶需求。隨著 Twitter/X 的使用者增多，對於輕量級客戶端的需求也隨之上升。

## 適合誰使用

**目標受眾**：喜歡在命令行中操作 Twitter 的開發者和使用者。

> [!example] 使用場景
> - 一名開發者用它來快速查看自己的 Twitter 時間線，因為他不想打開瀏覽器。
> - 一位學生用它來管理書籤，因為他希望能在終端機中快速找到有用的資訊。
> - 一名社交媒體經理用它來監控品牌的提及，因為他需要快速反應。

> [!warning] 注意事項
> 功能較為簡單，可能不支持所有 Twitter 功能。

> **Read:**
> - Timeline: fetch `for-you` and `following` feeds
> - Bookmarks: list saved tweets from your account
> - Search: find tweets by keyword with Top/Latest/Photos/Videos tabs
> - Tweet detail: view a tweet and its replies
> - List timeline: fetch tweets from a Twitter List
> - User lookup: fetch user profile, tweets, likes, followers, and following
