# twitter-cli

**1.4k** stars · **277** stars/天 · 建立 5 天前 · Python · Apache-2.0

`easy-install`

> [!summary] 一句話摘要
> 從終端讀取 Twitter 動態，方便用戶操作。

## 專案簡介

twitter-cli 讓用戶可以在終端中讀取 Twitter 的動態、書籤和用戶資料，無需 API 金鑰。它使用 Python 實現，並提供結構化輸出。與其他社交媒體 CLI 工具相比，twitter-cli 專注於 Twitter 的特定功能，提供更符合用戶需求的操作體驗。這是一個方便的工具，特別適合喜歡在終端操作的 Twitter 用戶。

**技術棧**：`Python`

## 重點功能

- 支持時間線、書籤和用戶查詢功能。
- 提供推文詳情和回覆查看功能。
- 結構化輸出可導出為 YAML 或 JSON 格式。

## 為什麼值得關注

> [!tip] 爆紅原因
> 開發者對多個社交平台有深入了解，這個專案滿足了對 Twitter 內容的需求，特別是在開發者社群中。

## 適合誰使用

**目標受眾**：喜歡在終端操作的 Twitter 用戶。

> [!example] 使用場景
> - Twitter 用戶 用它來 瀏覽最新動態，因為可以快速獲取信息。
> - 開發者 用它來 自動化推文分析，因為能夠方便地從終端操作。
> - 內容創作者 用它來 追蹤熱門推文，因為能快速了解趨勢。

> [!warning] 注意事項
> 僅支持 Twitter 平台。

> ## English
> 
> ### Features
> 
> **Read:**
> - Timeline: fetch `for-you` and `following` feeds
> - Bookmarks: list saved tweets from your account
> - Search: find tweets by keyword with Top/Latest/Photos/Videos tabs
> - Tweet detail: view a tweet and its replies
> - List timeline: fetch tweets from a Twitter List
> - User lookup: fetch user profile, tweets, likes, followers, and following
> - Structured output: export any data as YAML or JSON for scripting and AI agent integration
> - Optional scoring filter: rank tweets by engagement weights
> - Structured output contract: [SCHEMA.md](./SCHEMA.md)
> 
> > **AI Agent Tip:** Prefer `--yaml` for structured output unless a strict JSON parser is required. Non-TTY stdout defaults to YAML automatically. Us
