## 專案簡介

Garden Skills 是一個針對 AI 代理（如 Claude Code、Cursor 和 Codex）設計的技能集合，目的是讓開發者能夠快速整合多種功能。用戶可以透過 `npx skills add` 指令輕鬆安裝所需的技能，例如 `npx skills add ConardLi/garden-skills -s web-design-engineer` 來安裝網頁設計技能。這個專案的核心賣點在於其生產就緒的技能，這些技能不僅能提升開發效率，還能幫助用戶創建更具吸引力的前端作品。技術上，專案使用了 TypeScript 和 JavaScript，並且支援多種安裝方式，適合不同的開發環境，從 CI/CD 到本地開發皆可應用。

與其他類似工具相比，如 0x0funky/agent-sprite-forge 和 AlpinDale/parsync，Garden Skills 提供了更為全面的技能選擇，並且具備更高的可擴展性和靈活性，尤其在設計和影像生成方面。使用者在實際操作中可能會遇到依賴性問題，特別是在不同環境下的兼容性。這個專案目前仍在活躍開發中，適合中小型團隊或個人開發者使用，未來可能會持續增加更多技能和功能。對於需要快速集成多種 AI 能力的開發者來說，這是一個值得考慮的選擇。

**技術棧**：`TypeScript` · `JavaScript` · `CSS` · `HTML` · `Python`

## 重點功能

- 多種技能支持 — 包含網頁設計、影像生成和知識檢索等，滿足不同開發需求。
- 簡單安裝 — 使用 `npx skills add` 指令可快速安裝所需技能，支持多種安裝方式。
- 生產就緒 — 提供經過測試的技能，適合用於生產環境。
- 靈活的版本控制 — 支持從 GitHub Releases 下載固定版本，確保穩定性。
- 跨平台兼容性 — 能夠在多種 AI 代理環境中運行，提升使用靈活性。
