## 專案簡介

Open CoDesign 是一個開源的設計工具，旨在將用戶的提示轉換為精緻的原型、簡報或行銷資產，並且可以在本地運行。使用者只需提供一個 API 金鑰或使用 ChatGPT 訂閱登入，即可在 3 分鐘內開始創建作品。這個工具支持多種模型，包括 Claude、GPT、Gemini 和 Ollama，並且允許用戶將現有的 Claude Code 或 Codex 配置一鍵導入，省去繁瑣的設置過程。其最大的賣點在於完全本地運行，避免了雲端依賴和訂閱鎖定的問題，並且生成的設計文件會以 HTML、PDF、PPTX 等格式保存，方便用戶後續使用。技術上，Open CoDesign 是基於 Electron 開發的桌面應用，使用 TypeScript 和 Vue，並且支持多種平台（macOS、Windows、Linux）。

這個工具的設計理念是讓用戶在不受限於單一模型或雲端服務的情況下，靈活使用 AI 進行設計。與同類工具相比，如 Claude Design 和 Figma AI，Open CoDesign 提供了更高的自由度和數據隱私，因為所有資料都保存在用戶本地。使用者可以隨時查看生成過程，並在需要時進行干預，這在其他工具中並不常見。儘管目前有 48 個開放問題，但其解決率為 58%，顯示出社群的活躍度和維護者的回應速度。這個專案目前處於 beta 階段，適合小型團隊或個人開發者使用，未來可能會進一步增強功能和穩定性。

**技術棧**：`TypeScript` · `Vue` · `Electron`

## 重點功能

- 多模型支持 — 支持 Claude、GPT、Gemini、Ollama 等多種 AI 模型。
- 本地運行 — 完全在用戶的設備上運行，無需雲端服務。
- 即時生成 — 用戶可以在幾分鐘內從提示生成原型或簡報。
- 多種導出格式 — 支持 HTML、PDF、PPTX、ZIP 和 Markdown 格式的導出。
- 版本歷史管理 — 每個設計都有自己的工作空間，支持版本切換和歷史記錄。
