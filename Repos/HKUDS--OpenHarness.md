## 專案簡介

OpenHarness 是一個開源的代理架構，旨在為使用者提供一個強大的 AI 助手 Ohmo。使用者可以透過 CLI 指令 `oh` 啟動 OpenHarness，並整合各種工具和 API，讓 Ohmo 在 Slack、Telegram 和 Discord 等平台上運行。這個架構支援多種 API，包括 OpenAI 和 Claude，並能夠進行工具調用、技能加載和多代理協調。其核心特性包括流式工具調用、API 重試、並行執行和上下文記憶等，這些設計使得 Ohmo 能夠在長時間的會話中保持高效運作。

技術上，OpenHarness 使用 Python 和 TypeScript，並依賴多個庫如 `anthropic` 和 `openai`，確保了其功能的強大和靈活性。與其他工具相比，如 `0xGF/boneyard` 和 `AgentSeal/codeburn`，OpenHarness 提供了更為完整的代理架構和多平台支持，特別適合需要長期運行 AI 助手的開發者。使用者可以輕鬆配置和管理 API 提供者，並利用 `oh setup` 指令快速上手。這個專案目前處於 v0.1.7 版本，社群活躍度高，問題解決率達 85%。

**技術棧**：`Python 3.10` · `TypeScript` · `Docker`

## 重點功能

- 多平台支持 — 可以在 Slack、Telegram 和 Discord 上運行，讓使用者隨時隨地與 Ohmo 互動。
- 流式工具調用 — 支援並行執行工具，提升效率。
- 上下文記憶 — 能夠持久記錄會話歷史，讓 AI 助手在長時間會話中保持一致性。
- API 重試機制 — 具備指數回退的重試策略，確保穩定性。
- 可擴展的插件生態系統 — 支援自定義插件和技能，方便開發者擴展功能。

## 快速開始

1. 安裝 OpenHarness
```bash
curl -fsSL https://raw.githubusercontent.com/HKUDS/OpenHarness/main/scripts/install.sh | bash
```
2. 啟動 OpenHarness
```bash
oh
```
3. 設置 Ohmo 個人代理
```bash
ohmo init
```
