## 專案簡介

OpenWhip 是一個用於激勵 AI Claude 的工具，透過簡單的 CLI 指令來發送中斷信號和鼓勵訊息。用戶只需安裝後執行 `openwhip`，便可透過點擊系統托盤圖示來觸發 whip 行為，這會發送 Ctrl-C 信號並隨機顯示五種鼓勵訊息。這樣的設計讓用戶能夠輕鬆地與 AI 互動，並且可以在需要時快速激勵 AI 的表現。此工具支援 Windows、macOS 和 Linux，Linux 用戶需額外安裝 `xdotool` 以實現鍵盤自動化。核心功能包括簡單的 GUI 操控和即時反饋，讓用戶能夠在使用過程中獲得樂趣。這個工具的賣點在於其幽默感和輕鬆的使用方式，讓人們在開發過程中不會感到壓力，反而能以輕鬆的心情來激勵 AI。

**技術棧**：`Electron 33.0.0` · `Node.js >=18.0.0`

## 重點功能

- 簡易安裝 — 透過 `npm install -g openwhip` 安裝，支援多平台。
- 即時互動 — 點擊托盤圖示即可發送激勵訊息和中斷信號。
- 多種鼓勵訊息 — 隨機發送五種不同的鼓勵訊息，增加互動趣味。
- Linux 支援 — 需安裝 `xdotool` 以實現鍵盤自動化。
- 輕鬆開發 — 提供簡單的 CLI 指令，讓開發者快速上手。

## 快速開始

1. 全域安裝 OpenWhip
```bash
npm install -g openwhip
```
2. 啟動 OpenWhip
```bash
openwhip
```
3. 在 Linux 上安裝 xdotool
```bash
sudo apt install xdotool
```
