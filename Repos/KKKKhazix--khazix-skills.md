## 專案簡介

Khazix Skills 是一個針對 AI 研究和寫作的工具集合，包含輕量級的 Prompts 和結構化的 Skills。使用者可以直接複製 Prompts 到任何支持 AI 對話的模型中，或安裝 Skills 來自動化深度研究過程。這些 Skills 遵循 Agent Skills 的開放標準，能在 Claude Code、OpenClaw 和 Codex 等平台上使用。安裝過程簡單，使用者只需將所需的 Skill 文件夾複製到相應的 Skills 目錄下。這個工具的賣點在於其結構化的指令集和高效的研究框架，能夠快速生成高質量的研究報告。

技術上，Khazix Skills 主要基於 Python 開發，並且依賴於現有的 AI Agent 生態系統，這使得它能夠輕鬆整合到用戶的工作流中。與其他類似工具相比，如 0xGF/boneyard 和 MiniMax-AI/skills，Khazix Skills 提供了更為豐富的寫作和研究功能，特別適合需要深入分析和報告生成的使用者。在實際使用中，這些 Skills 可以顯著提高研究效率，尤其是在需要快速生成文檔的情況下。儘管目前仍在活躍開發中，社群的參與度和問題解決率顯示出這個專案的潛力。未來幾個月，預計會有更多功能和改進推出，適合對 AI 研究和寫作有需求的團隊使用。

**技術棧**：`Python`

## 重點功能

- Prompts — 提供多種通用深度研究框架，輕鬆複製到任何 AI 對話中使用。
- Skills — 遵循 Agent Skills 開放標準的結構化指令集，安裝後自動加載。
- hv-analysis Skill — 自動收集信息並生成排版精美的 PDF 研究報告。
- khazix-writer Skill — 包含完整的寫作風格規則和內容方法論，幫助用戶撰寫長文。
- 簡單的安裝方式 — 透過 Agent 或手動安裝，使用者可輕鬆整合到現有工作流中。

1. 克隆倉庫
```bash
git clone https://github.com/KKKKhazix/khazix-skills.git
```
