## 專案簡介

這個工具的核心功能是從微信 4.1 的 SQLCipher 加密數據庫中提取密鑰並解密聊天記錄。使用者需要在 macOS arm64 環境中運行，並禁用 SIP，然後透過 `find_key_memscan.py` 腳本提取密鑰，密鑰將儲存於 `wechat_keys.json`。接著，使用 `decrypt_db.py` 解密數據庫，最後可以透過 `export_messages.py` 將聊天記錄導出，支持多種查詢方式，如模糊匹配和關鍵字搜索。這個工具的賣點在於其簡單的命令行介面和對 AI 查詢的支持，讓使用者能夠直接從微信數據中獲取資訊。技術上，這個工具依賴於 Python 和 SQLCipher，並使用 LLVM 來進行內存掃描，這樣的設計使得提取密鑰過程相對高效。

與其他類似工具相比，如 ylytdeng/wechat-decrypt，這個專案專注於 macOS arm64 並提供了更直接的 AI 查詢接口。使用者可能會遇到的問題包括密鑰提取失敗或導出的數據中出現乱码，這些問題在熱門 Issues 中已有反映。整體來看，這個專案目前處於 beta 階段，適合對微信數據分析有需求的開發者，尤其是在需要集成 AI 查詢的情境下。未來可能會增加對更多版本的支持和改善導出數據的完整性。

**技術棧**：`Python` · `SQLCipher` · `LLVM`

## 重點功能

- 密鑰提取 — 使用 `find_key_memscan.py` 提取加密數據庫的密鑰，並保存至 `wechat_keys.json`。
- 數據解密 — 使用 `decrypt_db.py` 解密 SQLCipher 加密的微信數據庫。
- 聊天記錄導出 — 使用 `export_messages.py` 導出聊天記錄，支持多種查詢方式。
- 模糊匹配導出 — 支持通過聯絡人名稱模糊匹配導出聊天記錄。
- AI 查詢支持 — 集成 MCP Server，讓 AI 能直接查詢微信數據。

## 快速開始

1. 安裝依賴
```bash
brew install llvm sqlcipher
```
2. 提取密鑰
```bash
PYTHONPATH=$(lldb -P) python3 find_key_memscan.py
```
3. 解密數據庫
```bash
python3 decrypt_db.py
```
4. 導出聊天記錄
```bash
python3 export_messages.py --all
```

## README 摘錄

> [!info]- 展開查看原文 README
> # macOS 微信数据库解密
> 
> 提取微信 (WeChat) 数据库密钥，解密 SQLCipher 加密的本地数据库，导出聊天记录。支持 MCP Server，让 AI 直接查询微信数据。
> 
> ## 快速开始
> 
> ### 1. 前置条件
> 
> - macOS arm64，微信 4.x
> - 禁用 SIP：`csrutil disable`
> - 安装依赖：`brew install llvm sqlcipher`
> 
> ### 2. 提取密钥
> 
> 确保微信已登录并正在运行：
> 
> ```bash
> PYTHONPATH=$(lldb -P) python3 find_key_memscan.py
> ```
> 
> 密钥保存到 `wechat_keys.json`。
> 
> ### 3. 解密数据库
> 
> ```bash
> python3 decrypt_db.py
> ```
> 
> ### 4. 导出聊天记录
> 
> ```bash
> # 列出所有会话
> python3 export_messages.py
> 
> # 导出指定会话（支持模糊匹配联系人名）
> python3 export_messages.py -c "卡比"
> python3 export_messages.py -c wxid_xxx
> python3 export_messages.py -c 12345@chatroom
> 
> # 导出最近 N 条
> python3 export_messages.py -c "卡比" -n 50
> 
> # 搜索关键词
> python3 export_messages.py -s "关键词"
> 
> # 导出所有会话
> python3 export_messages.py --all
> ```
> 
> ### 5. MCP Server（让 AI 直接查询）
> 
> 安装依赖并注册到 Claude Code：
> 
> ```bash
> pip3 install fastmcp
> claude mcp add wechat -- python3 $(pwd)/mcp_server.py
> ```
> 
> 注册后 AI 可以直接调用以下能力：
> 
> | Tool | 功能 |
> |------|------|
> | `get_recent_sessions` | 获取最近会话列表 |
> | `get_chat_history` | 查看聊天记录（支持模糊匹配） |
> | `search_messages` | 跨会话搜索关键词 |
> | `get_contacts` | 搜索联系人 |
>
