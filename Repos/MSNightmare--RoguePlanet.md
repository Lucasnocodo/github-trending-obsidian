## 專案簡介

RoguePlanet 是一個針對 Windows Defender 的漏洞利用工具，主要利用了競爭條件（race condition）來實現系統權限提升。用戶在執行該工具後，若成功，將會獲得一個 SYSTEM shell。該漏洞在 Windows 10 和 Windows 11 的最新更新中經過測試，但不適用於 Windows Server，因為標準用戶無法掛載 ISO 映像。這個設計的選擇是基於 Windows 系統的普遍使用，尤其是在桌面環境中，因而能夠影響到大量用戶。該工具的「一句話賣點」是能夠在某些條件下實現 100% 的成功率，但這依賴於具體的系統環境和配置。

技術上，該工具使用 C++ 編寫，並且依賴於對 Windows 系統內部運作的深入理解，這使得它在安全研究中具有一定的價值。與其他類似工具相比，如 0x0funky/agent-sprite-forge 和 0xGF/boneyard，RoguePlanet 特別針對 Windows Defender 的漏洞，這使得它在特定環境下的利用效果更為顯著。實際使用中，成功率可能會受到系統配置的影響，並且需要用戶具備一定的技術背景來進行調整和測試。該工具目前處於早期階段，社群活躍度高，並且有多個開放的問題等待解決，這可能影響到未來的穩定性和功能擴展。總體來說，對於安全研究者和漏洞利用者來說，這是一個值得關注的工具，但在生產環境中使用時需謹慎考量其風險。
