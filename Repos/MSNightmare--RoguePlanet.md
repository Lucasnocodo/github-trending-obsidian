## 專案簡介

RoguePlanet 是一個針對 Windows Defender 的漏洞利用工具，利用競爭條件來實現系統權限提升。使用者只需在受影響的 Windows 10 或 Windows 11 系統上執行該 PoC，若成功，將會啟動一個 SYSTEM shell。這個漏洞的特點在於其成功率因機器而異，部分機器可達到 100% 的成功率，而其他則可能無法運作。該工具目前不支援 Windows Server，因為標準用戶無法掛載 ISO 映像，但開發者相信所有 Windows Server 版本也存在此漏洞，只是需要重新設計利用方式。這種設計選擇使得該工具在特定環境下的靈活性受到限制，但也顯示出其潛在的廣泛應用。

與其他漏洞利用工具相比，RoguePlanet 的競爭條件利用方式使其在某些情況下比靜態漏洞利用更具挑戰性和不確定性。這種不確定性在實際應用中可能導致使用者的挫折，特別是在需要穩定性的環境中。該工具目前處於活躍開發中，並且社群對其反應熱烈，儘管存在一些開放的問題和未解決的反饋。對於安全研究人員來說，這是一個值得關注的工具，尤其是在探索 Windows 系統漏洞的情境下。未來幾個月，隨著更多的測試和改進，該工具的成功率和穩定性有望提升。
