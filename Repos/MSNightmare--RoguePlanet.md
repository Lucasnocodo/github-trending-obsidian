## 專案簡介

RoguePlanet 是一個針對 Windows Defender 的漏洞利用工具，利用競爭條件漏洞來獲取系統權限。具體來說，這個工具會在特定條件下觸發漏洞，成功後會產生一個 SYSTEM shell。使用者需要在 Windows 10 或 Windows 11 的環境中測試，並且在某些機器上成功率可達 100%，但在其他機器上則可能失敗。這是因為漏洞的成功與系統的具體狀態有關，這使得它的應用具有一定的不確定性。該專案的核心賣點在於它能夠在不需要高級權限的情況下，利用 Windows Defender 的漏洞來獲取系統控制權。

技術上，這個工具使用 C++ 實現，並且依賴於對 Windows 系統的深入理解，特別是關於 ISO 映像掛載的行為。與其他類似的漏洞利用工具相比，如 0x0funky/agent-sprite-forge 和 0xGF/boneyard，RoguePlanet 的獨特之處在於它專注於 Windows Defender，並且能夠在特定條件下達成高成功率。實際使用中，這個工具可能會受到 Windows Server 的限制，因為標準用戶無法掛載 ISO 映像，這使得該漏洞的利用在伺服器環境中變得複雜。這個專案目前處於活躍開發中，社群對其反應熱烈，並且有多個相關的討論和提案。對於想要探索 Windows 安全性或進行滲透測試的開發者來說，這是一個值得關注的工具。
