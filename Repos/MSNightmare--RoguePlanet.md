## 專案簡介

RoguePlanet 是一個針對 Windows Defender 的漏洞利用工具，利用了競爭條件來獲取系統權限。使用者只需在受影響的 Windows 10 或 Windows 11 系統上執行 PoC，若成功，將會產生一個 SYSTEM shell。這個工具的特點在於其成功率因機器而異，某些機器能達到 100% 的成功率，而其他則不然。這是因為競爭條件的特性，可能需要針對不同環境進行調整。該工具在 Windows Server 上無法直接使用，但開發者相信所有 Windows Server 版本同樣存在漏洞，只需重新設計 PoC。這樣的設計使得 RoguePlanet 在特定情況下能夠有效利用系統漏洞，並且在漏洞利用的過程中，使用者需具備一定的技術背景來調整和測試。

相較於其他類似工具，如 0x0funky/agent-sprite-forge 和 0xGF/boneyard，RoguePlanet 專注於 Windows Defender 的特定漏洞，並且在使用上需要更高的技術能力。這使得它在安全研究和滲透測試中有其獨特的應用場景。使用者可能會遇到的問題包括成功率不穩定和需要針對不同環境進行調整，這可能會影響其在實際測試中的效率。該專案目前處於活躍開發階段，社群活躍度高，對於安全研究者來說，這是一個值得關注的工具。整體來看，RoguePlanet 適合有經驗的安全研究者和滲透測試者使用，並且在未來可能會有更多的功能更新和改進。
