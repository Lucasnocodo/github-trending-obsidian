## 專案簡介

RoguePlanet 是一個針對 Windows Defender 的漏洞利用工具，利用競爭條件來達成系統權限提升。使用者在執行此工具時，若成功，將會獲得一個 SYSTEM shell。該漏洞在 Windows 10 和 Windows 11（包括 Canary 版本）上經過測試，但不適用於 Windows Server，因為標準用戶無法掛載 ISO 映像。這個設計選擇是因為 Windows Server 的安全性限制，雖然開發者認為所有 Windows Server 版本也存在此漏洞，但未能在 PoC 中實現。工具的成功率因機器而異，有時可達 100%，但也有時會失敗，這是由於競爭條件的特性。這個工具的賣點在於其簡單的使用方式和直接的結果，對於安全研究人員和滲透測試者來說，提供了一個快速的權限提升方法。

技術上，這個工具使用 C++ 實現，並依賴於 Windows 的內部機制來觸發漏洞。相較於其他滲透測試工具，RoguePlanet 的特點在於其針對特定漏洞的精準性，而非通用的攻擊方法。這使得它在特定的環境中非常有效，但在其他情況下可能會遇到限制。使用者需要注意，這個工具的有效性會受到操作系統版本和配置的影響，並且在使用過程中可能會遇到一些技術挑戰，例如競爭條件的變化。整體而言，這是一個針對特定漏洞的專用工具，適合有經驗的安全研究人員使用。
