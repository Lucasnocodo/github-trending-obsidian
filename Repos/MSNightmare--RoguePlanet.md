## 專案簡介

RoguePlanet 是一個針對 Windows Defender 的漏洞利用工具，利用了競爭條件的漏洞來獲取系統權限。具體來說，這個工具會在 Windows 10 和 Windows 11 的環境中運行，當成功時會產生一個 SYSTEM shell。這個 exploit 的成功率因機器而異，部分機器可達到 100% 的成功率，而在其他機器上則可能失敗。這種設計的原因在於競爭條件的特性，開發者認為如果重新設計 PoC，可能能在所有條件下達到更高的成功率。此工具已在最新的 Windows 更新中進行測試，但不支援 Windows Server，因為標準用戶無法掛載 ISO 映像。

這一點是設計上的一個限制，但開發者相信 Windows Server 版本同樣存在漏洞，只需重新設計 exploit 即可。這個工具的賣點在於其能夠在特定環境中有效利用 Windows 的安全漏洞，並且對於安全研究人員來說，這是一個有趣的挑戰。使用者需要注意，這個 exploit 的成功依賴於多種因素，包括系統配置和環境變數。社群的活躍度顯示出對該工具的興趣，儘管目前存在一些未解決的問題，但開發者的回應速度相對較快。整體來看，這是一個針對特定漏洞的實驗性工具，適合對 Windows 安全漏洞有興趣的研究者和開發者。
