## 專案簡介

RoguePlanet 是一個針對 Windows Defender 的漏洞利用工具，利用競爭條件來獲取系統權限。這個工具的核心機制是透過創建一個競爭條件，當 exploit 成功時，會啟動一個 SYSTEM shell。使用者需要在 Windows 11 或 Windows 10 上運行，並且已安裝 2026 年 6 月的補丁。值得注意的是，這個 PoC 在 Windows Server 上無法運行，因為標準用戶無法掛載 ISO 映像，但作者相信所有 Windows Server 版本也是脆弱的，只是需要重新設計 exploit。這種設計的原因在於，競爭條件的特性使得 exploit 的成功率在不同機器上有所不同，作者提到在某些機器上可以達到 100% 的成功率。

相較於其他漏洞利用工具，RoguePlanet 專注於 Windows Defender，這使得它在特定環境下的有效性更高，但也限制了其適用範圍。使用者在實際操作中可能會遇到 exploit 不穩定的情況，這是因為競爭條件的本質。這個專案的社群活躍度相對較高，開放的問題和反饋顯示出使用者對於漏洞利用的興趣。整體來看，這個工具適合對 Windows 系統安全有深入了解的安全研究人員或滲透測試者，但對於一般用戶來說，使用風險較高。未來六個月內，若有更多的社群貢獻，可能會針對 exploit 的穩定性進行改進。
