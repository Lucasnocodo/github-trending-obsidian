## 專案簡介

RoguePlanet 是一個針對 Windows Defender 的漏洞利用工具，利用了系統中的競爭條件來實現權限提升。使用者只需執行 PoC，若成功，將會獲得 SYSTEM 權限的 shell。該工具已在 Windows 10 和 Windows 11 的最新版本上進行測試，但不支援 Windows Server，因為標準用戶無法掛載 ISO 映像。這個設計選擇反映了對於不同 Windows 版本的兼容性考量，雖然作者認為所有 Windows Server 版本也存在此漏洞，但未能提供有效的 PoC。該工具的核心賣點在於其簡單易用的 PoC，並且在某些機器上達到了 100% 的成功率。技術上，這個漏洞利用依賴於對競爭條件的精確控制，這使得其在不同環境下的表現可能會有所不同。

與其他類似工具相比，如 0x0funky/agent-sprite-forge 和 0xGF/boneyard，RoguePlanet 的特點在於其針對 Windows Defender 的專注，並且提供了具體的操作步驟來重現漏洞。實際使用中，使用者可能會遇到環境差異導致的成功率變化，這是該工具的一個限制。該專案的社群活躍度高，開放的問題數量相對較少，顯示出開發者對於問題的快速回應。這個專案目前處於 alpha 階段，適合安全研究人員和開發者進行漏洞測試和研究，但不建議用於生產環境。預測未來幾個月，該專案可能會進一步完善 PoC，並可能針對 Windows Server 提供新的解決方案。
