## 專案簡介

RoguePlanet 是一個針對 Windows Defender 的漏洞利用工具，利用競爭條件來獲取系統權限。使用者只需在受影響的 Windows 10 或 Windows 11 系統上執行該 PoC，若成功，將會產生一個 SYSTEM shell。這個工具的設計重點在於其不穩定性，因為競爭條件的特性使得成功率在不同機器上有所不同，某些機器可達到 100% 的成功率，而其他則可能失敗。這種不確定性是其主要挑戰之一。該工具已在 Windows 10 和 Windows 11 的最新更新上進行測試，但在 Windows Server 上無法正常工作，因為標準用戶無法掛載 ISO 映像。

這意味著雖然 Windows Server 也存在漏洞，但需要重新設計 PoC 才能利用。這種設計的靈活性和不確定性使得該工具在安全研究中具有一定的價值，但也增加了使用的風險。使用者需要具備一定的技術背景來理解和應用這個工具，特別是在漏洞利用和系統安全方面的知識。整體而言，RoguePlanet 是一個針對特定環境的專用工具，適合安全研究人員和漏洞利用專家進行深入研究。

**技術棧**：`C++`
