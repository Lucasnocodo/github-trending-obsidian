## 專案簡介

RoguePlanet 是一個針對 Windows Defender 的漏洞利用工具，利用競爭條件來獲取系統權限。使用者只需在受影響的 Windows 10 或 Windows 11 系統上執行此 PoC，若成功，將會啟動一個 SYSTEM shell。這個工具的設計考量了不同機器的成功率，雖然有時候成功率可能會有所不同，但在某些機器上可達到 100%。該工具的核心在於對競爭條件的利用，這意味著它依賴於系統在特定時間內的行為，這種設計使得漏洞利用的成功率受到環境的影響。與其他類似工具相比，RoguePlanet 的獨特之處在於它專注於 Windows Defender 的特定漏洞，而不是通用的權限提升技術。

這使得它在針對 Windows 環境時具有優勢，但在 Windows Server 上的使用受到限制，因為標準用戶無法掛載 ISO 映像。實際使用中，若 PoC 成功，將會獲得完整的系統訪問權限，這對於滲透測試或安全研究非常重要。這個專案目前仍在活躍開發中，儘管有一些開放的問題，但社群的反應相當熱烈。對於安全研究者來說，這是一個值得關注的工具，尤其是在針對 Windows 環境的漏洞挖掘方面。未來可能會有更多的改進和擴展，特別是在提升成功率和兼容性方面。
