## 專案簡介

RoguePlanet 是一個針對 Windows Defender 的漏洞利用工具，利用了競爭條件（race condition）來獲取系統權限。這個工具的核心機制是透過特定的操作使得系統在處理 ISO 映像時出現競爭條件，從而使攻擊者能夠獲得 SYSTEM 權限。使用者只需執行這個 PoC，若成功，將會啟動一個 SYSTEM shell。這個工具已在 Windows 10 和 Windows 11 的最新版本中進行測試，但不適用於 Windows Server，因為標準用戶無法掛載 ISO 映像。這個漏洞的成功率因機器而異，部分機器可達到 100% 的成功率，而其他機器則可能無法成功。這樣的設計使得 RoguePlanet 在不同環境中表現不一，增加了其使用的挑戰性。

這個工具的開發者 MSNightmare 似乎對於進一步改進 PoC 的可能性感到疲憊，並表示未來可能會有更好的設計來提高成功率。與其他類似工具相比，RoguePlanet 的獨特之處在於其專注於 Windows Defender 的特定漏洞，而不是一般的權限提升漏洞。這使得它在針對 Windows 環境的攻擊中具有特定的應用場景。使用者需要注意，這個工具的使用可能會引發法律和道德上的問題，因此應謹慎使用。整體來看，RoguePlanet 是一個針對特定漏洞的專業工具，適合於安全研究和滲透測試，但不建議在生產環境中使用。
