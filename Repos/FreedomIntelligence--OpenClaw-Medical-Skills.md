## 專案簡介

OpenClaw Medical Skills 是一個包含869種醫療AI技能的開源庫，專為 OpenClaw 和 NanoClaw 框架設計。使用者可以透過簡單的指令，如 `cp -r skills/* ~/.openclaw/skills/`，將技能安裝到全局或特定工作區，並且不需要重啟系統。這些技能涵蓋臨床報告、基因組學、藥物發現等多個領域，能夠將一般的AI代理轉變為專業的醫療研究助手。技術上，這個庫使用了 Git LFS 來管理大型數據文件，確保用戶可以選擇性地下載所需的技能，避免不必要的數據浪費。

與其他醫療AI工具相比，如 0x0funky/agent-sprite-forge 和 0xGF/boneyard，OpenClaw 提供了更為全面的技能集合，並且能夠直接連接到實際的數據庫和API，這使得它在實際應用中更具優勢。使用者在安裝過程中可能會遇到 Git LFS 的配置問題，這需要額外的注意。這個專案目前處於活躍開發中，社群活躍度良好，對於需要在醫療領域進行AI應用的團隊來說，這是一個值得考慮的選擇。未來六個月內，預期會有更多技能的加入和功能的擴展，進一步提升其在醫療AI領域的應用潛力。

**技術棧**：`Python` · `Jupyter Notebook` · `Docker`

## 重點功能

- 869種專業醫療技能 — 涵蓋臨床、基因組學、藥物發現等多個領域。
- 支持 Git LFS — 允許用戶選擇性下載大型數據文件，節省存儲空間。
- 自動加載技能 — 安裝後無需重啟，技能會自動在下一次會話中可用。
- 多種安裝方法 — 提供多種安裝選項，包括全局安裝和選擇性安裝。
- 與實際數據庫連接 — 能夠直接查詢 PubMed、FDA 等數據庫，提供實時資訊。

## 架構分析

OpenClaw Medical Skills 採用模組化設計，每個技能都是獨立的模組，這樣的設計使得技能可以靈活地被加載和卸載。資料流方面，技能透過 API 連接到外部數據庫，並能夠生成結構化的醫療或科學輸出。選擇這種架構的原因在於能夠快速擴展和更新技能，然而這也意味著需要定期維護和更新外部連接的穩定性。這個架構在處理大量數據時可能會遇到性能瓶頸，特別是在高併發查詢的情況下。整體而言，這種設計提供了良好的靈活性，但在擴展性上仍需謹慎考量。

> ## What Is This?
> 
> **OpenClaw Medical Skills** is a curated collection of **869 AI agent skills** covering the full spectrum of biomedical and clinical research. These skills are designed for [OpenClaw](https://github.com/openclaw/openclaw) / [NanoClaw](https://github.com/qwibitai/nanoclaw) — Claude-based personal AI assistant frameworks — and transform a general-purpose AI agent into a powerful medical and scientific research companion.
> 
> Each skill is a self-contained module (a `SKILL.md` file) that:
> - Teaches the agent specialized domain knowledge and workflows
> - Connects to real databases, APIs, and computational tools
> - Produces structured, clinically or scientifically relevant outputs
> 
> > We benefit from the open-source community. The full collection of resources can be found here: [Awesome LLM Resources](https://github.com/WangRongsheng/awesome-LLM-resources?tab=readme-ov-file#%E6%8A%80%E8%83%BD-Skills)
