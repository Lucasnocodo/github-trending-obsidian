## 專案簡介

OpenClaw Medical Skills 是一個包含 869 種醫療和科學研究技能的開源庫，旨在為 OpenClaw 和 NanoClaw 提供強大的醫療AI能力。用戶可以通過簡單的命令將技能安裝到工作區或全局環境，並且這些技能能夠連接到真實的數據庫和API，生成結構化的臨床或科學輸出。這些技能涵蓋了從臨床報告到基因組學、藥物發現等多個領域，讓一般的AI代理變成專業的醫療助手。技術上，這個庫使用 Python 和多種其他語言，並依賴 Git LFS 來管理大型數據文件。與其他醫療AI工具相比，OpenClaw Medical Skills 提供了更全面的功能，因為它整合了來自 12 個以上開源技能庫的資源，並且支持多種醫療數據查詢和分析。使用者可以在不需要重啟的情況下，輕鬆加載新技能，這在快速變化的醫療環境中非常重要。這個專案的成熟度還在提升中，社群活躍度良好，對於需要快速實現醫療AI功能的開發者來說，這是一個值得考慮的選擇。

**技術棧**：`Python` · `Jupyter Notebook` · `Docker`

## 重點功能

- 869 種技能 — 涵蓋臨床報告、基因組學、藥物發現等多個領域。
- 即時安裝 — 使用 `openclaw plugins install` 命令快速安裝單個技能。
- 支持 Git LFS — 管理大型數據文件，避免不必要的下載。
- 自動加載 — 新技能在下次會話中自動加載，無需重啟。
- 多語言支持 — 使用 Python、R、Shell 等多種語言編寫技能，靈活性高。

## 快速開始

1. 將技能安裝到工作區
```bash
cp -r skills/* /skills/
```
2. 全局安裝技能
```bash
cp -r skills/* ~/.openclaw/skills/
```
3. 使用 OpenClaw CLI 安裝單個技能
```bash
openclaw plugins install
```
4. 配置額外目錄
```bash
echo '{ "plugins": { "local": ["/path/to/OpenClaw-Medical-Skills"] } }' > ~/.openclaw/openclaw.json
```

OpenClaw Medical Skills 採用模組化設計，每個技能都是一個獨立的模組，這樣的設計使得技能可以輕鬆地被添加或移除。技能的資料流從用戶請求開始，經過技能模組處理後，連接到實際的數據庫或API，最後生成結構化的輸出。這種設計的好處是靈活性高，能夠快速適應不同的需求，但代價是需要用戶具備一定的技術背景來配置和使用。擴展性方面，隨著技能數量的增加，可能會對性能造成影響，特別是在大型數據集的處理上。

> ## What Is This?
> 
> **OpenClaw Medical Skills** is a curated collection of **869 AI agent skills** covering the full spectrum of biomedical and clinical research. These skills are designed for [OpenClaw](https://github.com/openclaw/openclaw) / [NanoClaw](https://github.com/qwibitai/nanoclaw) — Claude-based personal AI assistant frameworks — and transform a general-purpose AI agent into a powerful medical and scientific research companion.
> 
> Each skill is a self-contained module (a `SKILL.md` file) that:
> - Teaches the agent specialized domain knowledge and workflows
> - Connects to real databases, APIs, and computational tools
> - Produces structured, clinically or scientifically relevant outputs
> 
> > We benefit from the open-source community. The full collection of resources can be found here: [Awesome LLM Resources](https://github.com/WangRongsheng/awesome-LLM-resources?tab=readme-ov-file#%E6%8A%80%E8%83%BD-Skills)
