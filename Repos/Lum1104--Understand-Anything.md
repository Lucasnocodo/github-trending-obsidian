## 專案簡介

Understand Anything 是一個 Claude Code 插件，旨在幫助開發者理解大型代碼庫。使用者只需執行 `/understand` 命令，系統便會通過多代理管道分析整個項目，提取所有文件、函數、類別及其依賴關係，並生成一個知識圖譜，儲存於 `.understand-anything/knowledge-graph.json`。接著，使用者可以透過 `/understand-dashboard` 命令打開互動式網頁儀表板，視覺化代碼庫的結構，並可點擊任意節點查看其代碼和關係，還有簡單易懂的解釋。這樣的設計使得即使是初學者也能快速上手，了解代碼的運作方式。技術上，該工具結合了 LLM 智能與靜態分析，能夠提供即時的代碼解釋和架構導覽。

相較於傳統的代碼閱讀方式，這種方法能夠顯著減少理解代碼的時間，並提升團隊的協作效率。與其他工具相比，如 AlpinDale/parsync 和 FreedomIntelligence/OpenClaw-Medical-Skills，Understand Anything 提供了更深入的代碼關聯性分析和可視化功能，特別適合於大型代碼庫的探索和學習。使用者在實際操作中可能會遇到一些問題，例如大型圖譜無法完全顯示，這在 Issue #8 中已被提及。整體而言，這個工具適合於需要快速了解和導航複雜代碼庫的開發者和產品經理，尤其是在新團隊中。

**技術棧**：`TypeScript 5.7.0` · `React 18` · `Vite` · `TailwindCSS v4`

## 重點功能

- 互動式知識圖譜 — 代碼庫的文件、函數和類別以可視化方式呈現，點擊任意節點可查看其代碼和關聯。
- 簡單易懂的總結 — 每個節點都有 LLM 提供的解釋，讓非技術人員也能理解其功能。
- 引導式導覽 — 自動生成的架構導覽，按依賴順序學習代碼庫。
- 模糊和語意搜索 — 可依名稱或意義查找任何內容，支持語意查詢。
- 影響分析 — 在提交代碼前查看改動影響的部分，了解代碼的連鎖反應。
- 適應性 UI — 儀表板根據使用者角色調整顯示細節，適合初級開發者或高級用戶。
- 層級可視化 — 自動按架構層級分組，並以顏色標示不同層級。
- 語言概念 — 12 種編程模式在上下文中解釋，隨處可見。

## 架構分析

Understand Anything 採用多代理架構，通過五個專門的代理協同工作。首先，`project-scanner` 代理負責發現文件並檢測語言和框架。接著，`file-analyzer` 代理提取函數和類別，生成圖譜節點。

然後，`architecture-analyzer` 代理識別架構層級，`tour-builder` 生成引導學習導覽，最後 `graph-reviewer` 驗證圖譜的完整性。這種設計使得文件分析能夠並行進行，支持增量更新，僅重新分析有變更的文件。選擇這種架構的原因在於提高分析效率和準確性，但也可能增加系統的複雜性。

整體而言，這種設計能夠有效處理大型代碼庫，但在擴展性上可能會面臨挑戰，特別是在處理極大規模的項目時。

> Understand Anything is a [Claude Code](https://docs.anthropic.com/en/docs/claude-code) plugin that analyzes your project with a multi-agent pipeline, builds a knowledge graph of every file, function, class, and dependency, then gives you an interactive dashboard to explore it all visually. > ## 🤔 Why?
> 
> Reading code is hard. Understanding a whole codebase is harder. Documentation is always out of date, onboarding takes weeks, and every new feature feels like archaeology.
> 
> Understand Anything fixes this by combining **LLM intelligence** with **static analysis** to produce a living, explorable map of your project — with plain-English explanations for everything.

> ## 🔧 Under the Hood
> 
> ### Multi-Agent Pipeline
> 
> The `/understand` command orchestrates 5 specialized agents:
> 
> | Agent | Role |
> |-------|------|
> | `project-scanner` | Discover files, detect languages and frameworks |
> | `file-analyzer` | Extract functions, classes, imports; produce graph nodes and edges |
> | `architecture-analyzer` | Identify architectural layers |
> | `tour-builder` | Generate guided learning tours |
> | `graph-reviewer` | Validate graph completeness and referential integrity |
> 
> File analyzers run in parallel (up to 3 concurrent). Supports incremental updates — only re-analyzes files that changed since the last run.
> 
> ### Project Structure
> 
> ```
> understand-anything-plugin/
>   .claude-plugin/  — Plugin manifest
>   agents/          — Specialized AI agents
>   skills/          — Skill definitions (/understand, /understand-chat, etc.)
>   src/             — TypeScript source (context-builder, diff-analyzer, etc.)
>   packages/
>     core/          — Analysis engine (types, persistence, tree-sitter, search, schema, tours)
>     dashboard/     — React + TypeScript web dashboard
> ```
>
