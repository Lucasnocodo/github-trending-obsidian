## 專案簡介

HyperAgents 是一個自我參照的自我改進代理系統，旨在優化各種可計算任務。使用者需要在 .env 檔中設定 API 金鑰，並透過 Docker 環境來運行。安裝過程中，使用者需安裝 Python 3.12 及相關依賴，然後建立虛擬環境並安裝 requirements.txt 中的依賴。運行時，使用者可以透過 `python generate_loop.py --domains` 指令來啟動代理，並將結果輸出至 `outputs/` 目錄。這個工具的賣點在於其自我改進的能力，能夠在多個計算任務中持續優化性能。技術上，該專案依賴於 Python 及 Docker，並使用了一系列的庫，如 `requests`、`matplotlib` 和 `gym`，以支援強化學習和數據處理。

這些選擇使得 HyperAgents 能夠靈活應對不同的任務需求。與其他類似工具相比，如 `OpenAI` 的強化學習框架，HyperAgents 提供了更高的自我優化能力，能夠在執行過程中不斷調整策略。實際使用中，使用者可能會遇到實驗日誌無法下載的問題，這是因為缺少 LFS 物件。該專案目前處於早期階段，活躍度尚可，但仍需解決多個開放的問題。對於需要進行自我學習和優化的團隊，HyperAgents 提供了一個有趣的解決方案，但在生產環境中使用時需謹慎考量其安全性和穩定性。

**技術棧**：`Python 3.12` · `Docker` · `requests 2.32.4` · `matplotlib 3.10.3` · `gym 0.23.0`

## 重點功能

- 自我改進代理 — 能夠在執行過程中不斷優化策略，適應不同的計算任務。
- 多任務支持 — 可以同時處理多個可計算的任務，提升效率。
- Docker 支援 — 提供 Dockerfile 以簡化環境設置，方便部署。
- 實驗日誌管理 — 支援將實驗日誌存儲為多部分 ZIP 檔案，便於管理和分析。
- API 整合 — 可與多個 AI 模型的 API 整合，擴展功能。

## 快速開始

1. 安裝必要的系統依賴
```bash
sudo dnf install -y python3.12-devel graphviz graphviz-devel cmake ninja-build bzip2-devel zlib-devel ncurses-devel libffi-devel
```
2. 建立虛擬環境並安裝依賴
```bash
python3.12 -m venv venv_nat && source venv_nat/bin/activate && pip install -r requirements.txt
```
3. 建構 Docker 容器
```bash
docker build --network=host -t hyperagents .
```
4. 設定初始代理
```bash
bash ./setup_initial.sh
```
5. 運行 HyperAgents
```bash
python generate_loop.py --domains
```
