## 專案簡介

HyperAgents 是一個自我改進的代理系統，專為優化任何可計算任務而設計。使用者需設定 API 金鑰，並透過 Docker 環境來運行，這樣可以確保所有依賴都被妥善管理。安裝過程中，使用者需要安裝 Python 3.12 及相關的開發工具，然後創建虛擬環境並安裝需求包。核心功能包括生成循環的執行，並將結果輸出到指定的目錄。這個系統的賣點在於其自我改進能力，能夠在多次迭代中學習和優化性能，這對於需要不斷調整和改進的任務特別有用。技術上，HyperAgents 結合了多種 Python 庫，如 `gym` 和 `torch`，以支援強化學習和深度學習的應用。

這些選擇使得系統能夠在多種環境中運行，並且能夠處理複雜的計算任務。與其他類似工具相比，如 `0x0funky/agent-sprite-forge` 和 `NVIDIA/NemoClaw`，HyperAgents 提供了更高的靈活性和自我調整能力，特別是在自我改進的過程中。實際使用中，使用者需要注意安全性，因為執行的代碼可能來自於模型生成，存在一定的風險。這個專案目前仍在活躍開發中，適合對自我改進 AI 代理有需求的研究者和開發者。未來六個月內，隨著更多功能的加入，這個專案有潛力成為自我改進代理的領導者。

**技術棧**：`Python 3.12` · `Docker` · `gym 0.23.0` · `torch 2.1.0`

## 快速開始

1. 安裝必要的開發工具
```bash
sudo dnf install -y python3.12-devel graphviz graphviz-devel cmake ninja-build bzip2-devel zlib-devel ncurses-devel libffi-devel
```
2. 創建虛擬環境並安裝需求包
```bash
python3.12 -m venv venv_nat && source venv_nat/bin/activate && pip install -r requirements.txt
```
3. 構建 Docker 容器
```bash
docker build --network=host -t hyperagents .
```
4. 設置初始代理
```bash
bash ./setup_initial.sh
```
5. 運行 HyperAgents
```bash
python generate_loop.py --domains
```

## Star 趨勢

> [!abstract]- Stars 成長追蹤
> ```dataviewjs
> const me = dv.page("Repos/facebookresearch--HyperAgents");
> if (me?.star_history) {
>   const raw = me.star_history.toString();
>   const points = raw.split(",").map(p => { const [d, s] = p.split(":"); return { date: d, stars: parseInt(s) }; }).filter(p => !isNaN(p.stars));
>   if (points.length >= 2) {
>     const max = Math.max(...points.map(p => p.stars));
>     const lines = points.map(p => {
>       const w = Math.round(p.stars / max * 25);
>       return `${p.date} ${"\u2588".repeat(w)}${"\u2591".repeat(25-w)} ${p.stars.toLocaleString()}`;
>     });
>     const first = points[0].stars;
>     const last = points[points.length-1].stars;
>     const growth = first > 0 ? Math.round((last - first) / first * 100) : 0;
>     lines.push(`\n**成長** +${(last-first).toLocaleString()} stars（${growth}%）in ${points.length} snapshots`);
>     // 趨勢方向偵測
>     if (points.length >= 3) {
>       const mid = Math.floor(points.length / 2);
>       const fh = points.slice(0, mid), sh = points.slice(mid);
>       const rateF = fh.length > 1 ? (fh[fh.length-1].stars - fh[0].stars) / Math.max(1, (new Date(fh[fh.length-1].date) - new Date(fh[0].date)) / 86400000) : 0;
>       const rateS = sh.length > 1 ? (sh[sh.length-1].stars - sh[0].stars) / Math.max(1, (new Date(sh[sh.length-1].date) - new Date(sh[0].date)) / 86400000) : 0;
>       const ratio = rateF > 0 ? rateS / rateF : rateS > 0 ? 2 : 1;
>       const dir = ratio > 1.3 ? "Rising（加速中）" : ratio < 0.7 ? "Cooling（降溫中）" : "Stable（穩定）";
