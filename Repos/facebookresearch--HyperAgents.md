## 專案簡介

HyperAgents 是一個自我參照的自我改進代理系統，旨在優化各種可計算任務。用戶需要先設置 API 金鑰，然後安裝必要的依賴，接著透過 `python generate_loop.py --domains` 指令來運行代理。這個系統的設計重點在於其自我改進能力，能夠在執行過程中不斷調整和優化自身的行為。核心架構使用 Python 實現，並依賴 Docker 來簡化環境配置，這使得部署過程相對簡單，尤其是在多平台環境中。其依賴樹中包含了多個流行的 Python 庫，如 `requests` 和 `pandas`，這些都是數據處理和網絡請求的常用工具，顯示出其對數據驅動的設計理念。與其他類似工具相比，如 AlpinDale/parsync 或 BigBodyCobain/Shadowbroker，HyperAgents 的獨特之處在於其強調自我改進的能力，而不是僅僅執行預定任務。

這使得它在需要持續學習和適應的場景中更具優勢，例如在動態環境中的自動化任務。實際使用中，使用者可能會遇到如實驗日誌無法下載等問題，這在熱門 Issues 中被提及。這個專案目前處於活躍開發階段，社群互動尚可，但仍有許多開放的問題需要解決。對於希望在自動化和機器學習領域探索新方法的開發者來說，HyperAgents 提供了一個有趣的實驗平台。考慮到它的設計和功能，適合中小型團隊進行原型開發和研究，但在生產環境中使用時需謹慎評估其穩定性和安全性。

**技術棧**：`Python 3.12` · `Docker` · `requests 2.32.4` · `pandas 2.3.2`

## 重點功能

- 自我改進代理 — 能夠根據執行結果自我調整行為，提升任務執行效率。
- 多任務優化 — 支持多種可計算任務的優化，靈活應用於不同場景。
- Docker 支持 — 提供 Dockerfile 以簡化環境配置，方便在多平台上部署。
- 實驗日誌管理 — 提供實驗日誌的存儲和提取功能，便於後續分析。
- 安全考量 — 提供安全政策以約束代理的自我修改，降低潛在風險。

## 快速開始

1. 安裝依賴
```bash
sudo dnf install -y python3.12-devel graphviz graphviz-devel cmake ninja-build bzip2-devel zlib-devel ncurses-devel libffi-devel
```
2. 建立虛擬環境並安裝 Python 套件
```bash
python3.12 -m venv venv_nat && source venv_nat/bin/activate && pip install -r requirements.txt
```
3. 建構 Docker 容器
```bash
docker build --network=host -t hyperagents .
```
4. 運行代理
```bash
python generate_loop.py --domains
```
