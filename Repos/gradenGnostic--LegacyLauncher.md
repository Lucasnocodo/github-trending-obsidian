# LegacyLauncher

**296** stars · **49** stars/天 · 建立 6 天前 · JavaScript · MIT

> [!summary] 一句話摘要
> 為 Minecraft Legacy Console Edition 提供自定義啟動器，簡化遊戲啟動流程。

## 專案簡介

LegacyLauncher 是一個專為 Minecraft Legacy Console Edition 設計的自定義啟動器，提供了方便的 GUI 和自動更新功能。它使用 JavaScript 開發，支持跨平台運行，並允許用戶保存遊戲配置和個人資料。與其他啟動器相比，它提供了更符合 Minecraft 風格的界面和更靈活的配置選項，讓玩家能夠更輕鬆地管理遊戲。這是一個值得嘗試的工具，特別是對於喜愛 Minecraft 的玩家。

## 重點功能

- 提供 Minecraft 風格的 GUI，提升用戶體驗。
- 支持自動更新，確保用戶使用最新版本。
- 跨平台支持，兼容 Windows 和 Linux 系統。
- 允許用戶保存個人配置和遊戲資料，方便管理。
- 支持自定義啟動選項，靈活性高。

## 快速開始

從源碼安裝: `npm install`，然後運行: `npm start`

**目標受眾**：喜愛 Minecraft 的玩家和需要自定義啟動器的開發者。

> [!example] 使用場景
> - Minecraft 玩家 用它來 自定義啟動器，因為它提供了更符合遊戲風格的界面。
> - 開發者 用它來 測試不同的遊戲配置，因為可以輕鬆管理啟動選項。
> - 新手玩家 用它來 簡化遊戲啟動流程，因為界面友好且易於使用。

> [!warning] 注意事項
> 目前僅支持 Minecraft Legacy Console Edition，對於其他版本的支持有限。

> # LegacyLauncher
> 
> A custom launcher for Minecraft Legacy Console Edition.
> 
> ## Features
> 
> - **Minecraft-style GUI**: Authentic pixelated interface with Minecraft font and styling
> - **Automatic Updates**: Fetches latest releases from GitHub repositories
> - **Cross-platform Support**: Works on Windows and Linux
> - **Profiles**: Save your username and track playtime
> - **Custom Launch Options**: Configure IP, port, and server mode
> 
> ## Installation
> 
> ### From Source
> 1. Clone or download this repository
> 2. Install dependencies:
>    ```bash
>    npm install
>    ```
> 3. Run the launcher:
>    ```bash
>    npm start
>    ```
> 
> ## Building
> 
> ### Linux AppImage
> ```bash
> npm run dist
> ```
> 
> ### Windows Installer
> ```bash
> npm run dist:win
> ```
> 
> ## Configuration
> 
> ### Repository Source
> By default, the launcher fetches releases from `smartcmd/MinecraftConsoles`. You can change this in the Options menu.
> 
> ### Launch Options
> - **GitHub Repository**: Source repository for game releases
> - **Client Executable**: Name of the executable file (default: `Minecraft.Client.exe`)
> - **Compatibility Layer**: For Linux users - choose between direct execution, Wine, or Proton
> - **Connect/Bind IP**: Optional IP address for multiplayer
> - **Port**: Optional port number
> - **Server Mode**: Launch as headless server
> 
> ### Profile Settings
> - **Username**: Your in-game player name
> - **Playtime Tracking**: Automatically tracks total playtime
> 
> ## System Requirements
> 
> - **Windows**: Direct execution of Windows executables
> - **Linux**: Wine or
