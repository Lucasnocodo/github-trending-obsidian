## 專案簡介

ShadowBroker 是一個即時的多領域開源情報 (OSINT) 儀表板，將 60 多個即時情報來源融合於單一的黑色操作地圖介面。用戶可以追蹤飛機、船隻、衛星、衝突區域、CCTV 網絡等，所有數據實時更新。這個工具的設計理念是讓分析師、研究人員和無論是誰都能在同一地圖上看到全球的公共信號。使用者只需透過 Docker 部署，執行 `docker compose up -d` 即可啟動儀表板，無需收集或傳輸用戶數據，完全在本地運行。這樣的設計不僅提升了用戶的隱私，也簡化了部署流程。技術上，ShadowBroker 使用 Next.js 和 FastAPI，確保前端和後端的模組化架構，便於擴展和維護。

這個專案的賣點在於其即時性和多樣性，能夠整合來自不同來源的數據，並提供豐富的視覺化選項。與其他 OSINT 工具相比，如 OpenSky Network 和 Shodan，ShadowBroker 提供了更全面的數據覆蓋範圍，並且支持多種數據層的切換。使用者可以透過右鍵點擊地圖上的任意位置，獲取該國的資料和最新的衛星照片，這在其他工具中並不常見。儘管如此，這個工具仍然依賴於外部 API，使用者需要準備相應的 API 金鑰來獲取某些數據。整體而言，ShadowBroker 是一個功能強大且靈活的 OSINT 解決方案，適合需要即時情報的用戶。

**技術棧**：`Next.js` · `FastAPI` · `Python` · `Docker`

## 重點功能

- 即時數據整合 — 融合 60+ 個即時情報來源，提供全面的全球監控。
- 多層次視覺化 — 提供 35+ 可切換的數據層，支持不同的視覺模式（如衛星、夜光等）。
- 模組化架構 — 前端和後端可獨立擴展，便於維護和升級。
- 安全性設計 — 以限制 UID 運行容器，提升安全性。
- 無需用戶數據 — 儀表板完全在本地運行，無需收集或傳輸用戶數據。

## 架構分析

ShadowBroker 採用模組化架構，前端使用 Next.js 提供用戶界面，後端使用 FastAPI 處理數據請求。這樣的設計使得前後端可以獨立擴展，便於維護。數據流從多個外部 API 獲取，經過 FastAPI 處理後傳遞給前端顯示。

選擇 Docker 作為部署方式，能夠簡化環境配置和依賴管理，但也意味著需要用戶具備 Docker 的基本知識。這種架構的優勢在於能快速部署和更新，但在處理大量數據時可能會面臨性能瓶頸，特別是當多個數據源同時更新時。整體而言，這樣的設計使得 ShadowBroker 能夠靈活應對不同的使用場景。
