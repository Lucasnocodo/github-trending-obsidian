# TorchCode

**1.5k** stars · **252** stars/天 · 建立 6 天前 · Jupyter Notebook · 未標註授權

> [!summary] 一句話摘要
> 提供 PyTorch 面試準備的練習平台，實作核心操作。

> [!abstract] 核心創新
> 提供即時反饋的 PyTorch 練習平台，專注於面試準備。

## 專案簡介

這個專案讓用戶能夠從零開始實作 PyTorch 的核心操作，如 softmax 和注意力機制，並提供即時反饋。它基於 Jupyter Notebook，適合自我訓練或線上使用。與其他練習平台相比，它專注於深度學習的基本操作，幫助用戶鞏固基礎。這是一個非常適合 ML 工程師和學生的工具，值得一試。

**技術棧**：`Jupyter Notebook` · `Python` · `TypeScript` · `CSS` · `Dockerfile` · `Shell` · `Makefile` · `JavaScript`

## 重點功能

- 提供從零開始實作 PyTorch 核心操作的練習。
- 即時反饋，幫助用戶快速學習。
- 支持自我訓練或線上使用。
- 針對面試準備的專門設計。
- 涵蓋多種深度學習技術。

## 快速開始

1. 克隆倉庫
```bash
git clone https://github.com/duoan/TorchCode.git
```
2. 進入目錄
```bash
cd TorchCode
```
3. 啟動 Jupyter Notebook
```bash
jupyter notebook
```
