## 專案簡介

Pastel 是一款專為 macOS 26+ 設計的 IPA 下載工具，能夠輕鬆獲取舊版應用並自動捕獲數據包。用戶只需在 App Store 中搜索應用，選擇 Apple 帳戶地區，便可直接下載所需的 IPA 文件，並透過 AirDrop 安裝到 iPhone 或 iPad。這個工具的賣點在於其簡單的用戶界面和對多種語言的支持，讓用戶能夠快速找到所需的應用版本。技術上，Pastel 使用 SwiftUI 開發，完美適配 macOS 的 Liquid Glass 效果，並依賴 GSA 技術來處理雙重認證，這在其他 IPA 下載工具中往往是個痛點。與其他工具相比，如 ipatools 和 iMazing，Pastel 提供了更直觀的用戶體驗和更強的多語言支持，特別是在 Apple 生態系統中。實際使用中，Pastel 能夠處理多個應用版本的下載和刪除，並支持手動輸入 App ID 獲取最新版本。

儘管目前僅支持 Apple 硬體，但未來計劃推出 Windows 版本，這將擴大其用戶基礎。社群活躍度高，所有開放的問題均已解決，顯示出良好的維護狀態。這個專案目前處於 beta 階段，適合需要下載舊版應用的開發者和技術愛好者使用。若未來能夠支持更多平台，將會吸引更多用戶。對於小型團隊或個人開發者來說，這是一個值得嘗試的工具。

**技術棧**：`SwiftUI` · `Node.js`

## 重點功能

- 多語言支持 — 支持簡體中文、繁體中文、日語、韓語和泰語。
- 自動捕獲數據包 — 下載後可直接捕獲應用數據包，方便測試。
- AirDrop 安裝 — 下載的 IPA 文件可直接透過 AirDrop 傳輸至 iPhone 或 iPad。
- 手動輸入 App ID — 支持用戶手動輸入 App ID 獲取最新版本。
- 多選下載 — 支持多選應用版本進行下載和刪除。

## 架構分析

Pastel 採用 SwiftUI 作為前端框架，這使得其界面能夠完美適配 macOS 的 Liquid Glass 效果。後端使用 Node.js 處理數據請求和下載，這樣的選擇使得整體架構輕量且高效。數據流從用戶輸入 App ID 開始，經過 Node.js 伺服器獲取對應的版本 ID，然後將其返回給前端進行下載。

這種設計使得用戶能夠快速獲取所需的應用版本，並且能夠在多個來源中進行選擇。選擇 Node.js 作為後端技術，能夠快速處理請求，但在高並發時可能會成為性能瓶頸。整體架構的擴展性良好，但未來若需支持更多平台，可能需要重新考慮後端架構以適應不同的環境。

## README 摘錄

> [!info]- 展開查看原文 README
> # Pastel (迄今空前强大的一款 IPA 下载工具)。
> 一款用于安装 IPA 历史版本的工具，适用于获取旧版应用并自动捕获数据包。下载后，可直接通过 AirDrop 传输至 iPhone、iPad 上并安装并使用。
> 
> 目前只支持 macOS 26+ 且配备 Apple 芯片的 Mac。由于作者暂无 Windows PC 设备，故暂时没有 Windows 版本开发计划。
> 
> 使用 SwiftUI 编译，完美适配 macOS 26 的 Liquid Glass 效果。
> 
> # 主页面。
> 你可以轻松使用 Pastel 在对应地区的 App Store 内查看并搜索 App。更强大的是，它能直接根据你选择的 Apple 账户地区自动选择商店。在切换地区时，还会自动切换到你已登陆并对应地区的 Apple 账户。甚至支持直接下载一款此 Apple 账户从未下载过的 App。
> 
> # 现支持简体中文、繁体中文、日语、韩语和泰语作为主要语言。
> 
> # 空前强大的下载页面。
> 你可以直接在下载页面找到已下载的 IPA 文件，并直接预览它的 App Icon，这针对一些特定节假日，比如 618 期间打广告的 App 格外实用。点击分享按钮，便可直接 AirDrop 到你的 iPhone 或 iPad 进行安装。
> 
> # 初次使用。
> 初次使用，请前往“设置”>“Apple 账户”，添加你的 Apple 账户，一切数据都保存在 iCloud KeyChain，放心且安全。在登陆后完成双重认证，Pastel 会自动识别 Apple 账户的所属地区并完成登陆。解决了市面上已有 IPA 下载工具的很多问题，例如双重认证无法触发，经常丢失保存数据等 (使用了 GSA 技术来触发双重认证，更稳定且安全)。
> 
> # 众多来源可选。
> 云集了 Timbrd、Agsy、Bilin 下载源的版本 ID 信息，方便你快速找到对应的版本 ID。
> 
> 如果你想要从 Apple 获取，也没问题，只需要点击 Apple，它就会从你的 Apple 账户获取一次此 App 的版本 ID 并显示 (如果未曾拥有此 App，则会自动完成获取，付费 App 除外)。
> 
> 如果你知道对应 App 的版本 ID，也可手动输入版本 ID 下载。
>
