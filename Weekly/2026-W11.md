# 週報 - 2026-W11

> [!summary] 本週摘要

## 本週新增專案

```dataview
TABLE
  stars AS "Stars",
  stars_per_day AS "Stars/天",
  language AS "語言",
  category AS "分類",
  description AS "描述"
FROM "Repos"
WHERE week = "2026-W11"
SORT stars DESC
```

## 本週亮點

```dataview
TABLE
  stars_per_day AS "Stars/天",
  stars AS "Stars",
  category AS "分類"
FROM "Repos"
WHERE week = "2026-W11" AND stars_per_day >= 500
SORT stars_per_day DESC
```

## 分類分佈

```dataview
TABLE WITHOUT ID
  category AS "分類",
  length(rows) AS "數量",
  sum(rows.stars) AS "總 Stars"
FROM "Repos"
WHERE week = "2026-W11"
GROUP BY category
SORT length(rows) DESC
```

## 語言分佈

```dataview
TABLE WITHOUT ID
  language AS "語言",
  length(rows) AS "數量"
FROM "Repos"
WHERE week = "2026-W11"
GROUP BY language
SORT length(rows) DESC
```
